Hello everyone!<br><br>
The presented code is my first shot at building a javascript game.<br>
It is a two-players classic 8 ball pool game.

<h3>Keys:</h3>
* 'Q' : More shot power.<br>
* 'A' : Less shot power.<br>

<h3>Mouse:</h3>
* Left click : shoot.<br>
* Aim by moving the mouse.<br>
